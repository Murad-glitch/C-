📌 API Endpointlər
🟢 Product
GET /api/products
Bütün məhsulları gətir
POST /api/products
Yeni məhsul əlavə et
PUT /api/products/{id}
Məhsulu yenilə
DELETE /api/products/{id}
Məhsulu sil
🟢 Order
POST /api/orders
Yeni sifariş yarat
GET /api/orders
Bütün sifarişlər
GET /api/orders/{id}
Sifariş detalı
PUT /api/orders/{id}/status
Status dəyişmək
🟢 OrderItem
POST /api/orders/{orderId}/items
Sifarişə məhsul əlavə etmək
📌 Biznes Qaydalar (Vacib hissə)
1️⃣ Sifariş yaradılarkən status default olaraq "Pending" olmalıdır.
2️⃣ OrderItem əlavə ediləndə:
• Məhsulun qiyməti avtomatik Product-dan götürülməlidir.
• TotalAmount avtomatik yenilənməlidir.
3️⃣ Əgər Product IsAvailable = false-dursa sifarişə əlavə edilə bilməz.
4️⃣ Status dəyişmə qaydası:
• Pending → Preparing
• Preparing → Ready
• Ready → Completed
Geri qaytarmaq olmaz (basic versiyada).
